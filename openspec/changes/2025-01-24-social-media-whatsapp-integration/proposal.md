## Why
Customers increasingly order through WhatsApp and social media platforms, requiring a centralized system to manage all customer interactions and capture social media sales efficiently.

## What Changes
- Add comprehensive social media and WhatsApp integration to CRM
- Implement n8n-based workflow automation for message processing
- Create unified social inbox for all platforms
- Enable automatic order capture from WhatsApp messages
- Build social media analytics and attribution tracking

## Impact
- Affected specs: crm (new social integration capabilities)
- Affected code: src/app/admin/crm/, src/lib/services/crmService.ts, new API endpoints
- Dependencies: n8n instance, WhatsApp Business API, Facebook Developer account
