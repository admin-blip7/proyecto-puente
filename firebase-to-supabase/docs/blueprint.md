# **App Name**: Storefront Swift

## Core Features:

- Product Grid: Display products in a grid layout with name, image, and price.
- Shopping Cart: Allow users to add and remove products from a shopping cart.
- Real-time Updates: Display Real-time updates to the cart as items are added or removed
- Total Calculation: Calculate and display the total amount of the current sale, including any discounts or taxes (if applicable).
- Sales Summary: Generate a summary of the sale (e.g., items purchased, total amount) in a printable format for the customer. The AI tool will check to make sure that the local tax laws are respected, and suggest an addition if appropriate.
- Product Search: Enable searching for products by name or SKU.
- Admin Dashboard: Provide a dashboard to track daily sales and low-stock items.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) for a modern and trustworthy feel.
- Background color: Light gray (#F5F5F5), providing a clean and neutral backdrop.
- Accent color: Orange (#FF9933) to highlight calls to action and important information.
- Body and headline font: 'PT Sans', a humanist sans-serif, provides a modern look with a touch of warmth, suitable for both headings and body text.
- Use clear, simple icons from Material Design for navigation and product categories.
- Implement a responsive grid layout to adapt to different screen sizes.
- Incorporate subtle animations for transitions and user feedback.
